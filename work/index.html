Hazleton Pumps (SA)

Web Development & Digital Marketing

Hazleton Pumps is slurry pump manufacturer based in South Africa, exporting their unique award winning pump range to Africa and the rest of the world. This is another project work-in-progress taken on in steps to complete several small projects at a time. The previous website was created in WordPress but it was in this instance better to redesign the website to ensure that the layout and user experience was improved as well as optimizing for Search Engines. This site was created with Bootstrap and will be updated with a completed UI redesign once more user data and website content has been gathered. Currently we are busy implementing a Social Media strategy for Hazleton Pumps making use of Twitter and LinkedIn.
